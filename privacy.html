1. 개인정보 수집 및 이용

MOA 앱은 개인정보를 서버에 저장하지 않으며, 광고 및 분석을 위해 구글 서비스(AdMob, Firebase Analytics)를 사용합니다.

자동 수집 정보

광고 식별자 (Advertising ID)

수집 목적: 맞춤형 광고 제공  ·  제3자 제공: Google AdMob

앱 사용 패턴

수집 항목: 화면 조회, 버튼 클릭 등

수집 목적: 서비스 개선  ·  제3자 제공: Firebase Analytics

카메라 및 사진 접근 (v1.3.1 추가)

수집 목적: 선수 명단에서 이름 자동 인식(OCR)

처리 방식: 기기 내부에서만 처리, 서버 전송 없음

보관 기간: OCR 인식 즉시 삭제

제3자 제공: 없음

데이터 보관

  • 서버 저장: 없음
  • 기기 저장: 모든 팀/선수 데이터는 사용자 기기에만 저장
  • 삭제 방법: 앱 삭제 시 모든 데이터 삭제

2. 알림 기능

경기 결과 기록 리마인더를 위해 알림 기능을 사용합니다.

  • 알림 방식: iOS/Android Local Notifications (기기 내부에서만 생성)
  • 관리 방법: 기기 설정 → MOA → 알림

3. 권한 관리

사용자는 언제든지 다음 권한을 거부하거나 취소할 수 있습니다.

Android

설정 → 앱 → MOA → 권한: 카메라, 사진 및 동영상, 알림

iOS

  • 설정 → MOA: 카메라, 사진, 알림
  • 설정 → 개인정보 보호 → 추적: 광고 추적 거부

광고 ID 재설정

  • Android: 설정 → Google → 광고 → 광고 ID 재설정
  • iOS: 설정 → 개인정보 보호 → Apple 광고 → 개인 맞춤 광고 (OFF)
iOS 14.5 이상에서는 앱 추적 투명성(ATT) 정책에 따라 광고 추적 권한을 별도로 요청합니다.
